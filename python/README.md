## Type of variables

```
`int:` ціле число
`float:` число з рухомою комою
`chr:` окремий символ
`str:` рядок або послідовність символів
`bool:` булевий вираз, який може бути True (істинний) або False (хибний)
`NoneType:` Спеціальне значення (None), яке вказує на відсутність значення

```
